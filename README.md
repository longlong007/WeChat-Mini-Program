# Hello World 微信小程序教程

这是一个最简单的微信小程序示例项目，适合作为学习微信小程序开发的入门教程。

## 项目简介

本项目包含一个最基础的微信小程序，实现了一个简单的"Hello World"页面。通过学习本项目，你可以快速了解微信小程序的项目结构、配置文件和基本开发流程。

## 功能特点

- 最精简的代码结构
- 详细的代码注释
- 完整的教程文档
- 适合初学者入门

## 快速开始

### 1. 下载微信开发者工具

访问 [微信公众平台](https://mp.weixin.qq.com/) 下载并安装微信开发者工具。

### 2. 创建项目

1. 打开微信开发者工具
2. 点击"+"号创建新项目
3. 选择本项目所在的文件夹
4. AppID选择"使用测试号"
5. 项目名称填写"wechat_mini_demo"
6. 点击"创建"

### 3. 运行项目

创建项目后，微信开发者工具会自动编译并显示模拟器效果。你可以看到"Hello World!"的欢迎页面。

## 关键概念

### 1. 配置文件

微信小程序使用 JSON 格式的配置文件：
- `app.json`：全局配置
- `page.json`：页面配置
- `project.config.json`：开发者工具配置

### 2. 页面结构（WXML）

WXML 是一种类似 HTML 的标记语言：
- 使用 `view`、`text`、`button` 等组件
- 使用 `{{}}` 进行数据绑定
- 使用 `bindtap` 等绑定事件

### 3. 页面样式（WXSS）

WXSS 是一种类似 CSS 的样式语言：
- 使用 `rpx` 作为响应式单位
- 支持大部分 CSS 选择器
- 支持样式导入

### 4. 页面逻辑（JavaScript）

使用 Page() 函数注册页面：
- data：页面初始数据
- onLoad：页面加载时触发
- onShow：页面显示时触发
- onReady：页面渲染完成时触发

## 常见问题

### Q: 如何修改页面标题？

在页面的 `index.json` 文件中修改：

```json
{
  "navigationBarTitleText": "新标题"
}
```

### Q: 如何添加新的页面？

1. 在 `pages/` 目录下创建新页面文件夹
2. 创建 `index.js`、`index.json`、`index.wxml`、`index.wxss` 四个文件
3. 在 `app.json` 的 `pages` 数组中添加页面路径

### Q: 如何调试代码？

1. 使用微信开发者工具的 Console 面板查看日志
2. 使用 `console.log()` 输出调试信息
3. 使用 Sources 面板进行断点调试
