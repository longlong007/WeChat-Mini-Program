# 微信开发者工具使用指南

## 一、下载和安装

1. 访问微信公众平台：https://mp.weixin.qq.com/
2. 下载微信开发者工具
3. 安装并登录微信账号

## 二、创建新项目

### 1. 打开开发者工具

双击打开"微信开发者工具"应用程序。

### 2. 创建项目

```
1. 点击"+"号按钮
2. 填写项目信息：
   - 项目目录：选择代码所在文件夹
   - AppID：点击"使用测试号"（开发学习用）
   - 项目名称：填写项目名称
   - 开发语言：JavaScript
   - 模板选择：不使用模板
3. 点击"创建"按钮
```

## 三、开发者工具界面

### 1. 菜单栏

- **文件**：新建文件、打开文件、保存等
- **编辑**：撤销、重做、复制粘贴等
- **工具**：编译、预览、上传等
- **设置**：编辑器设置、代理设置等

### 2. 工具栏

```
┌─────────────────────────────────────────────────────┐
│  [编译] [预览] [上传] [项目] [...]                   │
└─────────────────────────────────────────────────────┘
```

- **编译**：编译当前项目
- **预览**：生成预览二维码
- **上传**：上传代码到微信服务器
- **真机调试**：连接真机进行调试

### 3. 编辑器区域

- **左侧**：项目管理器（文件列表）
- **中间**：代码编辑器
- **右侧**：模拟器窗口

### 4. 调试器区域

点击底部标签切换：
- **Console**：控制台日志
- **Sources**：源码调试
- **Network**：网络请求
- **Storage**：本地存储
- **AppData**：应用数据
- **Wxml**：页面结构
- **Sensor**：传感器调试

## 四、常用操作

### 1. 编译项目

```
方式一：点击工具栏的"编译"按钮
方式二：快捷键 Ctrl + B
方式三：自动编译（可在设置中开启）
```

### 2. 预览项目

```
1. 点击工具栏的"预览"按钮
2. 生成二维码
3. 使用微信扫描二维码
4. 在手机上演示效果
```

### 3. 真机调试

```
1. 连接手机（USB调试模式）
2. 点击工具栏的"真机调试"按钮
3. 在手机上进行操作
4. 在电脑上查看调试信息
```

### 4. 上传代码

```
1. 点击工具栏的"上传"按钮
2. 填写版本号和备注
3. 点击"上传"
4. 在微信公众平台提交审核
```

## 六、常见问题

### 1. 无法预览？

- 检查网络连接
- 确保微信账号已登录
- 检查是否开启了代理
- 尝试重新编译

### 2. 真机调试失败？

- 确保手机已开启USB调试
- 确认手机已安装微信
- 尝试重新连接USB
- 检查开发者模式是否开启

### 3. 编译报错？

- 检查语法错误
- 检查文件路径是否正确
- 检查JSON格式是否正确
- 查看Console日志定位问题

### 4. 样式不生效？

- 检查选择器是否正确
- 检查单位是否使用rpx
- 检查文件是否保存
- 尝试重新编译

## 八、调试技巧

### 1. Console日志

```javascript
// 普通日志
console.log('普通信息');

// 警告日志
console.warn('警告信息');

// 错误日志
console.error('错误信息');

// 表格形式
console.table([{a:1, b:2}, {a:3, b:4}]);
```

### 2. 断点调试

```
1. 在Sources面板中找到要调试的文件
2. 在代码行号左侧点击设置断点
3. 触发断点条件
4. 查看变量值和调用栈
```

### 3. Wxml调试

```
1. 点击"Wxml"标签
2. 查看页面DOM结构
3. 点击元素查看样式
4. 实时修改样式查看效果
```

### 4. AppData面板

```
1. 查看当前页面的所有数据
2. 实时修改数据查看效果
3. 观察数据变化
```
